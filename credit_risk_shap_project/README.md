
Credit Risk Modeling + SHAP Analysis
===================================

This project folder contains a complete deliverable matching the assignment screenshot located at:
/mnt/data/2dd66390-7a81-4697-99dd-c0b342d67b83.png

Files included:
- credit_data.csv         : synthetic sample dataset (replace with your real dataset if available)
- main.py                 : runnable pipeline (preprocess, train CV LightGBM, SHAP analysis)
- requirements.txt        : Python packages required
- report_generated.md     : concise text report summarizing pipeline and outputs
- selected_instances.csv  : three selected loan applications (high-risk, low-risk, borderline)
- shap_global_importance.csv : global feature importance produced by the pipeline
- shap_local_instances.json  : local textual explanations for the three instances
- credit_risk_shap_project.zip : this zipped project (created alongside these files)
